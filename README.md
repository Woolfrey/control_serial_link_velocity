## :floppy_disk: Installation

1. Clone the repository:

    `git clone https://github.com/Woolfrey/software_template.git`

2. Create the `build/` folder
   
   `cd software_template`
   
   `mkdir build && cd build`

4. Make the project

   `cmake ..`

   `make`
